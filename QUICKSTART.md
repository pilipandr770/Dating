# Быстрый старт

## Проект уже готов к запуску!

Все зависимости установлены. Осталось только запустить.

## Запуск за 2 минуты

### Терминал 1 - Backend:
```bash
cd backend
venv\Scripts\activate
python run.py
```

### Терминал 2 - Frontend:
```bash
cd frontend
npm run dev
```

### Откройте браузер:
```
http://localhost:3000
```

## Что уже готово:

- Backend структура (Flask API)
- Frontend структура (React + Vite)
- Все зависимости установлены
- База данных SQLite (создается автоматически)
- Конфигурационные файлы

## Что нужно сделать дальше:

1. Добавить реальные API ключи в `backend/.env`:
   ```
   STRIPE_API_KEY=ваш_ключ
   OPENAI_API_KEY=ваш_ключ
   ```

2. Добавить остальные файлы проекта из вашей предыдущей работы

3. Протестировать регистрацию и вход
